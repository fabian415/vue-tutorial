Git教學


  1. 安装Git，輸入命令 git --version，測試是否安裝成功？
  2. 移動到專案底下的資料夾，將專案資料夾建立成 git repository，輸入命令 git init
  3. 在專案下建立.gitignore檔，把不要上git的檔案填入
  4. 登入github網站 https://github.com/，建立一個新的專案叫vue-tutorial
  5. 輸入命令 git remote add origin git@gitlab.com:fabian415/vue-tutorial.git
  6. 輸入命令 git add .
  7. 輸入命令 git commit -m "Init project"
  8. 輸入命令 git push -u origin master

live-server install

  1. 首先需要安装Node.js，相信童鞋們都已經裝過了，没装的童鞋趕緊去下載安装。
  2. 打開命令列，輸入命令 npm i live-server -g
  3. 建置專案，輸入命令 npm init
  4. 然后輸入命令 live-sever 啟動

Homework 1:
  • 成品：To do list
  • 作業：下載
  • 解答：下載
  1. computed | computed | computed
  2. methods vs computed
    methods: 如果 data 內容如果一樣就還是會全部重算
    computed: 裡面有牽動到 data 才會跟著更換
    computed 無法帶參數，methods 可以
    不確定每次都會更新，請用 computed
    確定每次都會更新，就用 methods
  3. watch
  4. watch vs computed
    5. watch:
    * watch 需手動設置初始值
    * watch其相依的屬性有幾個，就得多幾個 watcher 監聽
    * 它的彈性讓你很容易針對資料異動設callback
  5. query API with lodash
    6. 總結：
    * computed: 適合處理一些數據格式，比如時間格式的轉換，字符串的截取等。
    * watch: 處理比較複雜的事件或是大邏輯，如允許我們執行異步操作（訪問一個API），限制我們執行該操作的頻率， 並在我們得到最終結果前，設置中間狀態。
    * methods: 要執行的函式，跟以上兩者無關。
